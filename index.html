Temperature vs. Latitude

The purpose of this project was to analyze how the weather change as you get closer to the equtor.To accomplish this project we first pulled data from (Openweather.com) API to assemble data on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs lattitude. Factors we looked at included: Temp, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
